# Open WebUI Projects

Open-source tools, filters, functions, and related utilities for Open WebUI.

## Tools

- [Advanced Tool Use](Tools/Advanced-Tool-Use/)  
  Tool Search + Programmatic Tool Calling for Open WebUI. Search across MCP,
  OpenAPI, and local tools on demand, then call one tool or run a sandboxed
  Python workflow that orchestrates many tool calls while returning only the
  final result to chat.

## Filters

- [Context Manager](Filters/Context-Manager/)  
  Deterministic prompt-context compaction for Open WebUI. Preserves system
  instructions and recent turns, compacts stale history into a retained summary,
  and drops raw older messages before the request reaches the model.
